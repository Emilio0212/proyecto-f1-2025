# 🏎️ F1 2025: El Fin de la Era Red Bull
**Autor:** Emilio Luevano Vazquez  
**Materia:** Visualización Gráfica para IA  

## 📌 Descripción del Proyecto
Este proyecto es una narrativa visual interactiva de "Data Storytelling" que analiza la temporada 2025 de la Fórmula 1. 
A través del análisis de datos de resultados de carreras, sprints y campeonatos, el proyecto responde a la pregunta de investigación: 
*¿Fue la temporada 2025 el año del cambio de guardia definitivo en la Fórmula 1, donde el trabajo en equipo superó al talento individual?* 
Se emplean visualizaciones interactivas para demostrar el cambio de dominio de Red Bull a McLaren.

## 💻 Instrucciones para ejecutar el proyecto localmente
Si deseas correr este proyecto en tu propia computadora, sigue estos pasos en tu terminal de comandos:

1. **Clona este repositorio:**
   git clone https://github.com/Emilio0212/proyecto-f1-2025.git

2. **Navega a la carpeta del proyecto:**
   cd proyecto-f1-2025

3. **Instala las dependencias necesarias:**
   (Se recomienda usar un entorno virtual).
   pip install -r requirements.txt

4. **Ejecuta la aplicación de Streamlit:**
   streamlit run app.py
